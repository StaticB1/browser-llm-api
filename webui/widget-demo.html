Browser·LLM

Embeddable chat widget — live demo

This is an ordinary web page. The floating chat bubble in the bottom-right was added with a single <script> tag and talks to your local Browser LLM server.

How it was added to this page

<script src="http://localhost:8081/widget.js"
        data-title="Ask AI"
        data-greeting="Hi! I'm running on a browser-driven LLM. Ask me anything."></script>

The widget discovers the API base from its own script URL, so the host page can live on any origin/port (CORS is open). Configure with data-* attributes: provider, title, accent, position (br/bl), greeting, system, open.

Try it

  • Click the 💬 bubble to open the chat.
  • Pick a provider in the header dropdown (gemini / chatgpt).
  • Ask for something with code — it comes back as a real code block.
  • Press Esc or the ✕ to close.

Control it from the page console too: BrowserLLMWidget.open(), .close(), .reset(), .config({accent:'#9b8cfb'}).
